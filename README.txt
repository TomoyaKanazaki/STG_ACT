◆ 作品説明 ◆
雑魚敵を倒すことでボスを怯ませて撃破するタイムアタック3Dアクションシューティングゲーム

◆ 工夫した点 ◆
◇ 当たり判定
	ベクトルの外積を利用して近接攻撃の当たり判定を正確にとれるようにしたこと。
	同じくっベクトルの外積を利用して床の判定を正確な正八角形でとれるようにしたこと。

◇ 敵の配置
	デバッグビルドの際にのみ表示されるGUIによって敵の出現を設定できるようにしたこと。
	GUIから直接出力できるため、データの安全性が高く編集のハードルが高いバイナリーファイルを活用したこと。

◆ 反省点 ◆
◇ "面白さ"の欠如
	企画立案の時点で実装したい処理から考えてしまったため、ゲーム性が薄くなってしまった。

◇ スケジュール管理
	当たり判定の作成に大幅に時間を取られてしまいスケジュール管理に失敗してしまった。

◆ 操作方法 ◆
[ キーボード / マウス ]
WASDキー		→ プレイヤーの移動
スペースキー	→ 射撃攻撃
右クリック		→ 斬撃攻撃
ESCキー			→ 実行プログラムの終了
